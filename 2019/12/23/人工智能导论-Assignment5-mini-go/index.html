Posted on 2019-12-23

#Mini_AlphaGo

阅读assignment5的代码做的一些阅读笔记。

###关于一些python库、函数的记录
1.go.py中有这样的代码N=int(os.environ.get('BOARD_SIZE',19))，STFW发现是获取环境变量的函数（不太懂），只要知道N=19就是了。这是关于os.environ.get的链接os.environ.get用法。

2.coords.py中出现了divmod函数，其作用是把除数和余数运算结果结合起来，返回一个包含商和余数的元组(a // b, a % b)。
e.g.>>>divmod(7,2)
   (3,1)

3.filter()过滤序列，过滤掉不符合条件的元素，返回由符合条件元素组成的新列表。filter(function,iterable)

4.set()创建一个无序不重复元素集，可进行关系测试，删除重复数据，还可以计算交集、差集、并集等。class set([iterable]),set()

5.numpy居然还可以用元组作为索引。

6.List的pop方法移除列表最后一个元素并返回其值。

7.collections.namedtuple()方法类似于C里面的struct。

8.time.time()返回当前的时间戳。

###围棋环境environment阅读
####coords.py
1.这个比较简单，就是介绍了棋盘坐标的四种表示方式minigo coord、flat、SGF、GTP，以及坐标之间的转化。我们框架代码中用的是minigo coord，即一个二元数组(a,b),0<=a,b<=18

####go.py
1.棋盘board用19×19的numpy数组表示，每个位置存的是颜色，黑、白、空分别用1，-1，0表示，这样交换颜色的话可以直接对数组取负。玩家动作是(Color,Move)的元组。而Move是(Coordinate c|None)。

2._check_bound(c)是检查二元组c有没有越过棋盘边界的。

3.place_stones(),应该是在棋盘s处下上颜色为color的棋。

4.replay_position(),对一个位置的历史动作均重新玩一次，每次返回新的位置。

5.里面许多函数的参数c其实是位置坐标(line,column)。

6.find_reached()函数返回的chain是包含状态c的列表，reached是包含当前位置c上下左右四个位置与其颜色不同的位置的列表。
